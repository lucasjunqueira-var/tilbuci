Resources used on development

 

Developing TilBuci is a huge task. This software creaton wouldn't be possible without the usage of the folllowing resources from our free software community. I'd like to thank a lot their development teams for the kind gesture of making such great tools available to the world!

 

HAXE

 

Haxe can build cross-platform applications targeting JavaScript, C++, C#, Java, JVM, Python, Lua, PHP, Flash, and allows access to each platform's native capabilities. Haxe has its own VMs (HashLink and NekoVM) but can also run in interpreted mode.

 

haxe.org

 

OPENFL and LIME

 

From enterprise applications to best-selling games, your creativity can be expressed with OpenFL. Publish to native C++ on desktops, phones, or tablets, or deploy to the web with JS and WebGL. Do it all using one seamless toolset.

 

openfl.org

 

FEATHERS UI

 

Cross-platform user interface components for creative frontend projects.

 

feathersui.com

 

ELECTRON JS

 

Build cross-platform desktop apps with JavaScript, HTML, and CSS. Used for desktop application deployment.

 

www.electronjs.org

 

APACHE CORDOVA

 

Mobile apps with HTML, CSS & JS. Target multiple platforms with one code base. Free and open source. Used for mobile application deployment.

 

cordova.apache.org
